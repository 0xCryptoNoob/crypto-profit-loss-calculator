# Crypto Profit/Loss Calculator

This is a beginner Python console application that calculates the profit or loss from a cryptocurrency investment based on user input.

## Features
- Accepts investment amount and current value
- Calculates profit or loss
- Displays results rounded to two decimal places

## Technologies Used
- Python 3

## How to Run
1. Make sure Python is installed
2. Run the program:
   ```bash
   python profit_loss_calculator.py
